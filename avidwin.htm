Avid windows

 

The project window is the first one displayed. It has four buttons:

  • Bins displays all the bins which you have created as well as the trash icon if you have thrown anything away.

 

  • Settings is used to configure user preferences.

 

  • Info shows whether the video is PAL or NTSC.

 

  • New Bin allows you to create a new bin for the video. You can give your bins any name, to suit your project.

 

You will find several more windows within the application:

·         The composer, which is used both for checking the video which is being captured and for playout of the material edited so far.

·         The timeline normally includes at the bottom of the display, the timecode, which shows the duration of your edit.

·         You can see small icons at the left of the timeline: V1, A1, A2, A3 and A4. These represent video and audio channels.
