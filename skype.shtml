Skype - Chatting cheaply

A few days ago I was chatting with my brother without using a phone. I had recently downloaded Skype. It's a program which allows people to chat betwee each other around the world. For a little money, around 2 cents an hr, you can chat to people in various countries more cheaply than if you were using a normal landline.

In so doing I have been able to keep in touch with family in different places without strengthening my fingers through hours of typed messages on programs like MSN messenger, ICQ or others.

Particularly interesting with this program is that when MSN messenger and other programs fail to connect this one allows me to chat to friends with no troubles at all.
