Starting a project:

 

·         Doubleclick the AvidXpress icon on the desktop. When it opens click on the button displaying "new user"

·         A dialogue box appears where you can type your name

·         Repeat this process with the "Create new project" dialogue box. You will be given the choice between NTSC and PAL. In Europe PAL should be selected.

·         Once this is done highlight your user name as well as the name of your project.
